How we do it

When it comes to modding, it all starts by creating a 3D model of an object or, vehicle we want to create. For this step we use Maya from Autodesk. It's a great piece of a software for modeling Farm sim equipment.


Next step into modding is baking textures. Now this process depends whether you are making vehicles or objects. Software we are using is called Substance Painter.


Final piece of puzzle is in-gaming our model. Tools we use are usually Giants Editor (requires login) and Notepad++.
